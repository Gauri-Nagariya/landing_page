This is a fully responsive landing page for PureBrush, an oral care brand. The page includes a hero section, product highlights, best-sellers, client gallery, testimonials, features, and a subscription call-to-action. Built using React and Tailwind CSS, it features interactive elements such as image hover effects, scroll-responsive navbar, and responsive grid layouts for mobile and desktop devices.
